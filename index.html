MODOO RETURN 생태계

  1. 1. MODOO RETURN STABLE COIN.

    MODOO RETURN 코인은 스테이블 코인 이다. MODOO RETURN 프로젝트를 통해 구축될 MODOO RETURN 생태계를 유지하고 활성 화하기 위한 매개체 역할을 할 것입니다.
    MODOO RETURN 토큰은 어떠한 자산가격과도 관련되지 않으며, 유틸리티 토큰으로서 오로지 MODOO RETURN플랫폼에서만 가상화폐 구매수단으로 사용됩니다.
    또한, MODOO RETURN STABLE COIN은 투자한 금액과 같은 금액으로 MODOO RETURN 프로젝트에서 M마크 순금으로 교환이 가능합니다.

  2. 2.I'LL BE BACK COIN.

    IBBC 토큰은 MODOO RETURN 플랫폼에 사용이 가능하며, 금으로 교환이 가능 합니다. IBBC는 MODOO RETURN 플랫폼에서 거래 할 수 있는 토큰으로 1 IBBC = 금 1g 으로 고정됩니다.
    IBBC토큰은 MODOO RETURN STABLE COIN을 지급하는 수단으로 IBBC토큰을 스테이킹함으로써 투자한 금액의 500/1로 계산되어 매일 지급됩니다.

암호화폐와 실물자산의 연결

  • 관리 및 안전성

    암호화폐와 실물자산의 경계를 줄여 효율적인 거래 방법과 안전한 거래를 구축합니다.

  • 디지털화

    실물자산으로 거래 가능한 토큰화 된 암호화폐는 블록체인기반으로 안전하게 보장됩니다.

  • 다양성

    암호화폐 및 실물자산 금을 통해 포트폴리오를 구축하여 시장의 불확실성으로부터 자산을 안전하게 보호할 수 있습니다.

WITH PROOF-OF-OWNERSHIP ON THE BLOCKCHAIN

MODOO RETURN 통해 실물 금으로 교환할 수 있습니다.

MODOO RETURN 플랫폼은 이용자의 자산을 안전하게 저장하고 이동할 수 있는 기능을
제공하고, 거래의 동시성을 유도하여 중개자 없이도 상호 신뢰할 수 있는 거래가 가능하도록
하는 플랫폼입니다. 서비스 내부 DB에 저장되던 이용자의 자산을 서비스 외부 DB,
블록체인에 저장하여 이용자간, 매개 체간 이동이 가능하도록 설계된 플랫폼입니다.

플랫폼에서는 이용자 자산 가치의 척도화를 위해 MODOO RETURN TOKEN 화폐로 이용하며,
이 MODOO RETURN을 통해서 누구나 공정하고 안전한 Gold 거래를 진행할 수 있습니다.
